Создать Task Manager. Пользователь должен иметь возможность добавить задачу, которая состоит из:

имени пользователя;
е-mail;
текста задачи;
картинки;
предварительного просмотра (он должен работать без перезагрузки страницы).

Требования к изображениям - формат JPG/GIF/PNG, не более 320х240 пикселей. При попытке загрузить изображение большего размера, картинка должна быть пропорционально уменьшена до заданных размеров.

Сделать вход для администратора (логин "admin", пароль "123"). Администратор имеет возможность редактировать текст задачи и поставить галочку о выполнении. Выполненные задачи в общем списке выводятся с соответствующей отметкой.

Сделать фильтр “показать все задачи” / ”показать выполненные”.

В приложении нужно с помощью чистого PHP реализовать модель MVC (PHP-фреймворки использовать нельзя). Верстка на bootstrap.
